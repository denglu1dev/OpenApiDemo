# 准备工作
在进行登录易接入之前，需要在登录易开放平台注册开发者账号，并且添加好一个网站应用，获取相应的 AppId 之后（SDK 需要获取 AppSecret），可以开始接入流程。

开放平台注册地址
https://open.denglu.net.cn/web/pages/user-register.html

# 流程说明
第三方网站应用发起登录易服务授权请求，登录易用户允许操作后，登录易服务器将第三方网站应用的账号密码加密发送到其服务器，经由第三方网站应用服务器验证账号密码。验证成功后返回授权 token 和 cookie 给登录易服务器，登录易会重定向到第三方网站，并且带上由第三方应用授权 token 和 cookie。

注意，./util/rsa.php里面的密钥对需要更改，文件中的密钥对只是一个示例，为了安全起见，必须要改成自己企业自己生成的，公钥填入开放平台，秘钥则保存在自己保存起来。
